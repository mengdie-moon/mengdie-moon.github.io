  • C++ - 匿名函数lambda

    摆脱传统函数 来到匿名世界 等等 -- 寄语
    基本语法:
    [捕获列表](参数列表) mutable(可选) 异常属性 -> 返回类型 { // 函数体 }
    语法规则：lambda表达式可以看成是一般函数的函数名被略去，返回值使用了一个 -> 的形式表示。唯一与普通函数不同的是增加了“捕获列表”
    编译（要指定-std=c++11）：
    g++ -o lambda_test lambda_test.cc -std=c++11...
